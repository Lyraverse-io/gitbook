# Wallet Connection Guide

Learn how to securely connect your wallet to Lyraverse.

## Supported Wallets

### Recommended Wallets
- **MetaMask**: Most popular, full feature support
- **Coinbase Wallet**: User-friendly mobile experience
- **Trust Wallet**: Good mobile alternative
- **Any Hyperliquid-compatible wallet**

## Step-by-Step Connection

### 1. Install Wallet Extension
```bash
# MetaMask (Browser Extension)
Visit: https://metamask.io/download/
Install for Chrome/Firefox/Edge
```

### 2. Create or Import Wallet
- **New Wallet**: Follow setup wizard, save seed phrase securely
- **Import**: Use existing seed phrase or private key

> ⚠️ **Security Warning**: Never share your seed phrase or private keys!

### 3. Configure Hyperliquid Network

#### Manual Network Addition
```json
Network Name: Hyperliquid Testnet
RPC URL: https://api.hyperliquid-testnet.xyz/evm-rpc
Chain ID: 998
Currency Symbol: ETH
Block Explorer: https://explorer.hyperliquid-testnet.xyz/
```

#### Using WalletConnect (Mobile)
- Scan QR code from desktop
- Select Hyperliquid network in mobile wallet

### 4. Fund Your Wallet
- **Testnet HYPE**: Use Hyperliquid faucet
- **Mainnet**: Purchase HYPE on exchanges
- **Bridge**: Use official Hyperliquid bridge

### 5. Connect to Lyraverse
1. Visit [lyraverse.io](https://lyraverse.io)
2. Click **"Connect Wallet"** (top-right)
3. Select your wallet from popup
4. Approve connection request
5. You're connected! ✅

## Connection Troubleshooting

### "Wrong Network" Error
- Switch to Hyperliquid network in wallet
- Refresh the page
- Try reconnecting

### "Connection Failed" Error
- Check wallet extension is enabled
- Try different browser
- Clear browser cache
- Restart wallet extension

### Mobile Connection Issues
- Ensure WalletConnect is updated
- Try different mobile browser
- Check internet connection

## Security Best Practices

### Wallet Security
- ✅ Use hardware wallet for large amounts
- ✅ Enable 2FA when available
- ✅ Never share private keys
- ✅ Use separate wallet for testing

### Transaction Safety
- ✅ Always verify contract addresses
- ✅ Check transaction details before signing
- ✅ Start with small amounts
- ✅ Use testnet for experimentation

### Phishing Prevention
- ✅ Only connect to official sites
- ✅ Verify URLs carefully
- ✅ Don't click suspicious links
- ✅ Report suspicious activity

## Managing Multiple Wallets

### Adding Accounts
1. Open wallet extension
2. Click account dropdown
3. Select "Create Account" or "Import Account"
4. Switch between accounts as needed

### Switching Networks
1. Click network dropdown in wallet
2. Select "Hyperliquid Testnet" or "Hyperliquid"
3. Refresh Lyraverse page if needed
