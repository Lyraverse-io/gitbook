# Managing Your Collections

Learn how to manage, update, and monitor your NFT collections after creation.

## Accessing Collection Management

### Owner Dashboard
1. **Connect Wallet**: Ensure you're connected with collection owner wallet
2. **Go to Collection**: Navigate to your collection page
3. **Management Tab**: Click "Manage" in collection navigation
4. **Owner Verification**: Automatic ownership verification

### Available Actions
- View analytics and statistics
- Update collection settings
- Manage minting phases
- Configure DAO governance
- Access financial reports

## Collection Analytics

### Real-Time Metrics
- **Total Minted**: Current supply vs max supply
- **Floor Price**: Lowest active listing price
- **Volume**: Total trading volume
- **Holders**: Unique wallet addresses owning NFTs

### Time-Based Analytics
- **24h Volume**: Trading volume last 24 hours
- **7d Volume**: Weekly trading data
- **30d Volume**: Monthly performance
- **Price History**: Floor price evolution

### Holder Distribution
- **Top Holders**: Largest NFT owners
- **Holder Count**: Total unique owners
- **Concentration**: Ownership distribution metrics

## Updating Collection Settings

### Editable Parameters
- **Description**: Update collection description
- **External Links**: Website, social media
- **Banner Image**: Change hero image
- **Metadata**: Update off-chain data

### Immutable Parameters
- **Name**: Cannot be changed after creation
- **Symbol**: Fixed at deployment
- **Max Supply**: Cannot be increased
- **Creator Address**: Permanent record

## Phase Management

### Active Phase Monitoring
- **Live Metrics**: Real-time minting progress
- **Time Remaining**: Countdown to phase end
- **Mint Rate**: NFTs per minute
- **Error Tracking**: Failed transactions

### Phase Modifications
- **Price Adjustments**: Within active phases only
- **Time Extensions**: Add time to running phases
- **Limit Changes**: Modify per-wallet limits
- **Reserve Management**: Adjust reserved allocations

## Financial Overview

### Revenue Tracking
- **Primary Sales**: Direct minting revenue
- **Secondary Sales**: Marketplace royalties
- **Platform Fees**: Lyraverse fees deducted
- **Net Earnings**: Your total revenue

### Fee Structure
```
Minting Revenue: 100% to creator
Marketplace Sales: 97.5% to seller (2.5% platform)
Royalties: Creator percentage (0-25%)
Network Fees: Paid by users
```

## Troubleshooting

### Common Issues

**Cannot Access Management**
- Verify wallet connection
- Confirm ownership
- Check network selection

**Analytics Not Updating**
- Wait for indexer sync (~5 minutes)
- Refresh page
- Check network status

**Phase Changes Not Applying**
- Confirm within modification windows
- Check contract permissions
- Verify transaction confirmation
