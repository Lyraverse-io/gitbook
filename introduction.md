## Technology Stack

- **Blockchain**: Hyperliquid (high-performance L1)
- **Smart Contracts**: Custom minting and marketplace contracts
- **Frontend**: Next.js with TypeScript
- **Backend**: Node.js API with PostgreSQL
- **Indexing**: Custom blockchain indexer

## Platform Fees

- **Minting**: 2.5% platform fee on mint
- **Marketplace**: 0.5% platform fee on sales
- **Royalties**: Creator-defined (0-25% recommended)

## Security & Transparency

Lyraverse prioritizes security and transparency:
- All contracts are verified on Hyperliquid
- No admin keys for user funds
- Open-source smart contracts
- Community governance options
